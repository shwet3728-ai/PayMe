# PayMe

Smart Queue Management & Pickup Platform

## Features

* QR Ordering
* Digital Token System
* Real-time Order Tracking
* Shop Dashboard
* Admin Dashboard
* Razorpay Payments
* Push Notifications
* Multi-Shop SaaS

## Tech Stack

Frontend:

* Next.js
* TypeScript
* TailwindCSS

Backend:

* NestJS
* PostgreSQL
* Prisma ORM

Realtime:

* Socket.IO

Payments:

* Razorpay

Notifications:

* Firebase Cloud Messaging
